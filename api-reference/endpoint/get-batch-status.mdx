---
title: "Get Batch Live Call Status (Detailed)"
description: "Retrieves the live, individual status of *every call* within a batch."
---

Retrieves the live, individual status of *every call* within a batch. This is the primary endpoint for monitoring a campaign in real-time.

*   **Endpoint:** `GET /api/v1/status/batch/:batchId`
*   **Method:** `GET`
*   **URL Parameters:**
    *   `batchId` (string, required): The ID returned from `POST /api/v1/batch`.
*   **Success Response (200 OK):**
    Returns a `calls` array containing the status object for every call in the batch.
